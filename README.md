vim-min-ide
===========

VIM IDE settings. Not a complete IDE, but more than an editor

Installation
---------

Requirements
```
sudo apt-get install vim exuberant-ctags
```

Clone
```
git clone git@github.com:abyrcheg/vim-min-ide.git
```

Copy
```
cp ~/vim-min-ide/.vimrc ~
cp -r ~/vim-min-ide/.vim ~
```
